# Mock Clinic API

API for managing doctors, locations, and specializations.  
Built with Flask, SQLAlchemy, Postgres, and Docker.

### Prerequisites

- Docker

### Run the App

Start your Docker (Note: I use Colima to do this):
- `colima start`

Start the app with:
- `docker compose up --build`



## Tech Stack

- Flask
- SQLAlchemy
- Flask-Migrate (alembic)
- Postgres
- Docker
